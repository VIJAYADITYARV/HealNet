# 🌐 HealNet: The Future of Patient-Centric Clinical Intelligence

![HealNet Banner](https://raw.githubusercontent.com/VIJAYADITYARV/HealNet/main/client/public/logo-placeholder.png)

**Live Demo**: [https://healnet-henna.vercel.app](https://healnet-henna.vercel.app)

HealNet is a premium, high-fidelity MERN application designed to bridge the gap between complex medical data and patient understanding. By leveraging advanced **Generative AI (Gemini 1.5 Pro)** and a state-of-the-art **Neural Surface UI**, HealNet empowers users to own their health journey with confidence, clarity, and community support.

---

## ✨ Core Pillars of HealNet

### 🧠 1. AI Intelligence Lab
The central nervous system of HealNet. It transforms raw medical complexity into actionable patient knowledge.
- **Medical ID Core**: Encrypted, high-fidelity emergency medical profiles accessible via unique secure QR codes.
- **Report Intelligence**: Decipher medical jargon from uploaded lab reports into plain, supportive English.
- **Outcome Radar**: Predictive analysis of treatment success based on real-world community datasets.
- **Live Community Trends**: Real-time ticker tracking regional health alerts and treatment breakthroughs.

### 🛡️ 2. The Medical Vault (Health Profile)
A military-grade dashboard for personal health record management.
- **Dynamic CRUD Intelligence**: Secure management of medical logs, surgeries, and prescriptions.
- **Baseline Context**: Intelligent capture of chronic history, allergies, and daily check-ins.
- **Neural Pulse**: Visualizing health trends and vitals through interactive, high-contrast charts.

### 🤝 3. Patient Perspective Network (Community)
Healing is better together. We've built a zero-noise, high-trust network.
- **Verified Journeys**: Badge-verified recovery stories that increase trust and clinical validity.
- **Anonymous Perspectives**: Share sensitive health updates with complete privacy toggles.
- **Smart DM System**: End-to-end encrypted messaging with AI-powered conversation context and suggestions.

## 🛠️ Technical Architecture

### **Frontend**
- **Core**: React 19 + Vite (Fast Refreshes)
- **State**: Redux Toolkit (Global Identity Management)
- **Styling**: Tailwind CSS + Custom Design Tokens
- **Vibe**: Lucide-React for high-fidelity clinical icons

### **Backend**
- **Runtime**: Node.js + Express 5 (Latest)
- **Database**: MongoDB Atlas (Scalable JSON Records)
- **Intelligence**: Gemini 1.5 Pro AI Engine
- **Auth**: JWT (Stateless Secure Sessions)
- **File Handling**: Multer for high-resolution medical scan uploads

## 🚀 Quick Start (Local Setup)

### 1. Prerequisites
- Node.js (v18+)
- MongoDB connection string
- Google Gemini API Key

### 2. Installations
```bash
# Clone the repository
git clone https://github.com/VIJAYADITYARV/HealNet.git

# Install Server Dependencies
cd server
npm install

# Install Client Dependencies
cd ../client
npm install
```

### 3. Environment Config
Create a `.env` in the `server` folder:
```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_ultra_secure_secret
GEMINI_API_KEY=your_google_ai_key
PORT=5000
```

### 4. Run Development
```bash
# Terminal 1 (Server)
cd server
npm run dev

# Terminal 2 (Client)
cd client
npm run dev
```
